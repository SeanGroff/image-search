FCC Image Search

User stories:

  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • I can get a list of the most recently submitted search strings.

Example image search:

https://branch-squid.glitch.me/api/imagesearch/funny memes

Example Image Search with Pagination

https://branch-squid.glitch.me/api/imagesearch/funny memes?offset=2

View most recent search strings

https://branch-squid.glitch.me/api/latest/imagesearch
